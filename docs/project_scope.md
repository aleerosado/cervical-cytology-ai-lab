# Project Scope

## Current Scope

This project focuses on:

- AI-assisted cervical cytology analysis
- PAP smear image classification
- Explainable medical AI
- Research experimentation

---

# Current Limitations

The project does not currently include:

- Clinical deployment
- Real-world diagnosis
- Regulatory validation
- Hospital integration
- Patient data processing

---

# Intended Use

Research and educational purposes.

---

# Initial Technical Scope

Input:
- PAP smear cell images

Output:
- Cell classification
- Confidence score
- Explainability visualization

---

# Non-Goals

The following are currently outside the project scope:

- Full-slide pathology analysis
- FDA-grade diagnostic systems
- Real-time hospital systems
- Multi-modal patient analysis

---

# Long-Term Expansion Possibilities

- Histopathology
- Biomedical signal analysis
- Neurotechnology
- Cognitive biomarkers
- AI-assisted diagnostics
